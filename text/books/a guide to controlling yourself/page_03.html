Key Combinations


Examine(E) + Any Mouse Click: Examine

Grab(Ctrl) + Mouse1: Grab with Right Hand
Grab(Ctrl) + Mouse2: Grab with Left Hand

Grab(Ctrl) + Mouse1 (On Held Weapon): Wield with Right Hand
Grab(Ctrl) + Mouse2 (On Held Weapon): Wield with Left Hand

Throw(R) + Mouse1: Throw Right Object
Throw(R) + Mouse2: Throw Left Object

Drop(Q) + Mouse1: Drop Right Object
Drop(Q) + Mouse2: Drop Left Object

Quick Self(Z) + Mouse1: Activate Right Object
Quick Self(Z) + Mouse2: Activate Left Object

Alternative Action(Alt) + Mouse1: Right Object Alternative Action
Alternative Action(Alt) + Mouse2: Left Object Alternative Action

Double Tap Block(C) + Movement: Dodge in that direction (3 second cooldown).
Hold Block(C) + Any Mouse Click, Empty Hand: Dodge towards the target (3 second cooldown).
Hold Block(C) within 1 second of attack: Parry incoming damage (3 second cooldown).
